### What This Skill Does

This is a **complete operational SOP** for running an SEO/GEO Agent. It answers the question: *"I have an AI agent — how do I make it run my SEO day-to-day?"*

### The 4-Week Timeline

```
Week 0: Foundation (GSC + GA4 + API keys + baseline)
     ↓
Week 1: BOFU landing pages (pricing, comparisons → highest conversion)
     ↓
Week 2: Internal links + content distribution (index coverage jumps)
     ↓
Week 3: GEO reinforcement (FAQ Schema, AI-friendly format)
     ↓
Week 4: CTR optimization + scale (high-impression pages → clicks)
     ↓
Result: ~32K impressions, 8-11 keywords on Page 1, 86% index coverage
```

### Core Deliverables

| Deliverable | Frequency | Purpose |
|:------------|:----------|:--------|
| SEO Daily Report | Daily | Track "keywords on Page 1" as headline metric |
| Keyword → Landing Page Table | Continuous | Single source of truth — one keyword, one page |
| CTA Convert Blocks | Every page | Convert traffic to [CTA_GOAL] |
| Owner Checklist | One-time | Human tasks Agent can't do (auth, payments) |

### Key Innovation: GEO + SEO Unified

This SOP doesn't treat GEO as an afterthought. Every content piece is simultaneously optimized for:
- **Google ranking** (keywords, internal links, Schema)
- **AI citation** (direct answers, tables, FAQ, IndexNow push)
- **Conversion** (CTA blocks with UTM tracking)

### Tools Stack (Minimum Viable)

```
DataForSEO (paid) — keyword Volume/KD + real-time ranking + AIO detection
+ Google Search Console (free) — real impressions/clicks/CTR
+ GA4 (free) — traffic → conversion funnel
= Everything you need to run this SOP
```

## ❓ FAQ

**Q: What's the difference between `gingiris-seo-geo` and `gingiris-seo-geo-agent`?**
A: `gingiris-seo-geo` is the **strategy playbook** (what to do and why). `gingiris-seo-geo-agent` is the **execution SOP** (exactly what to do every day, with templates). Use both together: strategy informs the plan, this SOP executes it.

**Q: Can I really get 32K impressions in one month?**
A: Yes — verified with a real content site. The key drivers are: index coverage expansion (8→57 pages), long-tail keyword breadth (30+ tracked words), and dual-platform distribution. Your mileage depends on domain age, niche competition, and content velocity.

**Q: What's the minimum budget to run this?**
A: DataForSEO (~$50-100/month for basic ranking queries) + GSC (free) + GA4 (free) = ~$50-100/month total. Everything else is optional.

**Q: Does my Agent need special API access?**
A: Your Agent needs DataForSEO credentials for ranking data. GSC/GA4 require OAuth (human does this once in the Owner Checklist). The Agent handles everything else.

**Q: Is this only for English markets?**
A: No. The SOP is language-agnostic. Set `[GEO_TARGET]` to your market (US-en, ja, ko, etc.) and all keyword/ranking queries adapt automatically.
